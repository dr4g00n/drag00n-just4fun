## [1.0.0] - 2026-04-14

### Added

#### 核心功能
- ✨ 自动地图采集系统（房间、出口、NPC）
- 🔐 环境变量登录管理（安全存储凭据）
- 🔄 tmux 后台持久运行支持
- 📊 结构化 JSON 日志输出

#### 测试与文档
- 🧪 完整测试框架（Mock MUD Server + 自动化测试）
- 💊 数据胶囊系统（AI 友好的知识沉淀，6 个胶囊）
- 📚 开箱即用的 Examples（基础、高级地图、自定义登录）
- 📖 技术文章《Tintin++ 自动化最佳实践》

#### Python 工具
- 🐍 BFS 最短路径寻路引擎
- 🔍 NPC 查询定位系统
- 💾 地图数据后处理脚本

#### 仓库基础设施
- 📄 MIT License 开源协议
- 📝 完整 README.md
- 🐛 Issues 模板（Bug 报告、功能请求）
- 🤝 贡献指南

### Improvements
- 🔧 优化正则匹配性能
- 🎨 改进终端输出颜色高亮
- 📐 完善目录结构

### Documentation
- 📚 新增 `examples/` 目录（3 个示例文件）
- 📚 新增 `docs/` 目录（2 篇技术文章）
- 📚 新增 `CONTRIBUTING.md`（贡献指南）
- 📚 更新数据胶囊（6 个胶囊）

# 欢迎来到 Tintin++ MUD 自动化脚本项目！ 👋

欢迎来到 **drag00n-just4fun** 项目的社区！这里是讨论 Tintin++ 自动化、MUD 游戏开发、脚本编写的友好空间。

---

## 📢 项目简介

这是一个为书剑 MUD 打造的 Tintin++ 自动化解决方案，包含：
- 🗺️ 自动地图采集
- 🔐 安全登录管理
- 🔄 后台持久运行
- 🧪 完整测试框架
- 💊 数据胶囊系统

**仓库**：https://github.com/dr4g00n/drag00n-just4fun

【开源分享】Tintin++ MUD 自动化脚本 v1.0.0

大家好，我最近整理了一套 Tintin++ MUD 自动化脚本，现已开源分享。

核心功能：
- 自动地图采集（房间、出口、NPC）
- 环境变量安全登录
- tmux 后台持久运行
- 完整测试框架
- 丰富的 Examples（基础、高级地图、自定义登录）

仓库：https://github.com/dr4g00n/drag00n-just4fun

主要支持书剑 MUD，理论上兼容所有标准 MUD 协议服务器。

包含完整文档和测试框架，适合想学习 Tintin++ 自动化的朋友参考。

欢迎 Star / Fork / 提建议！

Title: Open Source Tintin++ Automation Script - v1.0.0 Released

Hi everyone,

I've been working on a Tintin++ automation script for MUD games and just released v1.0.0.

Features:
- Auto mapping (rooms, exits, NPCs)
- Secure login via environment variables
- Background execution with tmux
- Complete test framework
- Ready-to-use examples

GitHub: https://github.com/dr4g00n/drag00n-just4fun

Currently supports Chinese MUD servers but should work with any standard MUD protocol.

Includes full documentation and testing framework. Happy to accept contributions!

tags: #mud #tintin #automation #opensource
